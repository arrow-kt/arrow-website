# Arrow 2.2.3

Arrow 2.2.3 is out!
The main focus of this release is bugfixing — especially around Jackson serialization.
On top of that, we also make Arrow work better with the latest version of Kotlin.

**Improvements for Kotlin 2.4.**
- Added [`holdsIn` contracts](https://kotlinlang.org/api/core/kotlin-stdlib/kotlin.contracts/-contract-builder/holds-in.html)
  when useful. This means the compiler has additional information that may be used
  for further smart casting.
- Arrow's [atomic primitives](https://arrow-kt.io/learn/coroutines/concurrency-primitives/#atomic)
  are now implemented on top of Kotlin's [common atomics](https://kotlinlang.org/docs/whatsnew2120.html#common-atomic-types).

As a result of these changes, the minimal required version of Kotlin is now 2.2.0.
